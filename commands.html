Commandlist

List of bot commands

Command

Description

Usage

set-prefix

Set or update the command prefix for this server.

/set-prefix [new prefix]

commandlist

Get a list of all custom commands for this server

/commandlist

create-command

Create a new custom command for this server.

/create-command

remove-command

Remove a mentioned custom command from this server.

/remove-command [customcommand-name]

update-command

Change name, response or cooldown of the mentioned command.

/update-command [customcommand-name]

help

Get more information about the commands or individual commands.

/help [commandname]

invite

Get the inventation link to invite the bot to your server.

/invite

ping

Ping the bot to get Discord's Latency.

/ping
